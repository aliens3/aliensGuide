chapter0では，このガイドが何のために書かれたのかについて語ります． と言っても，答えはそのタイトルにすでに述べられています．すなわち，

我々ヒトの欲望の実現のために世界をコントロールする方法を学ぶ (A)

chapter1以降のすべての内容は，このために書かれていますので，この文Aの意味をchapter0で理解しましょう．

そこでこの文Aを，「我々ヒトの欲望の実現のために」と「世界をコントロールする」の2つに分けて，何を意味するのか，それぞれ見ていきましょう．

{% endsection %} {% section "homo_sapienes",'1|我々はヒトであるがゆえに，原理的に可能な欲望はすべて許されている' %}

まずは「我々ヒトの欲望の実現のために」からです． といっても，すべての欲望が実現できるわけではありません． そこで，「実現可能な欲望」とは何か，から見ていきましょう．

[imaOK実現できない欲望をもっていても意味がないからね]

{% subsection "homo_sapienes","1.1|実現可能な欲望であるためには3つの条件を満たす必要がある",'1|原理的に可能な欲望はすべて許されている' %}

「実現可能な欲望」とは何でしょうか？

まず，第一にそれは欲望でなくてはなりません．つまり，それを欲しないのなら，「欲望」ではないので「実現可能な欲望」にもなりません． それゆえ，例えば，火の中に手をつっこむという行為は，ほとんどのヒトにとって「実現可能な欲望」ではありません．

[imaOKでも僕は火の中に手をいれてみたいなあ・・・]

それが欲望であるという第一の条件を満たしていれば，第二にそれは実現可能でなくてはなりません． 実現可能であるためには2つの条件を満たす必要があります．

1つ目はそれは原理的にできるのか？ということです． 例えば，好きな人への決死の告白が失敗し，時を戻したい，と欲しても，時を戻すことは現代の物理学の知識では原理的にできません． だから，この欲望は実現可能ではありません．

2つ目は原理以外の点，例えば倫理や法律などにおいても可能か？ということです． 原理的にできるとしても，実現可能とは限りません． 例えば，あるヒトの受精卵が病気の原因となるような遺伝子をもっていたとしましょう． それを改変することで病気を予防することは，原理的にはできますが，実現可能ではありません． なぜなら，それは2022年版の人間倫理観に反する行為とみなされていますし，法律でも禁止されているからです．

以上から，実現可能な欲望であるためには， 第一にそれを欲するか，第二にそれは原理的に可能か，第三にそれは倫理・法律等から可能か，3つすべてでyesである必要があります．

[imaOK 人間の実現可能性の条件の図，あとで違いの図もつくる] {% endsubsection %} {% subsection "homo_sapienes","1.2|ヒトの欲望の実現可能性は原理的に可能かだけで決まる",'1|原理的に可能な欲望はすべて許されている' %}

しかし，そのような3つの条件を検討する必要があるのは誰なのでしょうか？ それは我々人間です．が，文Aをもう一度見るとわかるように，

我々ヒトの実現可能な欲望のために世界をコントロールする方法を学ぶ (A)

ガイドの対象は人間ではなくヒトなのです．

このように我々をヒトと書くのは，我々をヒト科ヒト属に分類される動物として扱いたいからです．

[imaOKチンパンジーやゴリラやヒトが横並びになっている・並列的な，アウストラロピテクスも薄くいれて] 正確には，我々はヒト属のうち，ホモサピエンスという種です． 過去にはヒト属にはネアンデルタール人とかホモ・エレクトス(代表例はジャワ原人や北京原人)などもいましたが， すべて絶滅し，現存するのはホモサピエンスだけです． ですので，ホモサピエンスという長い種名は使わずに簡単に属名で我々をヒトと呼ぶことにします．

一方，我々の存在は他の生物と並列に扱うことはできんのだ，というニュアンスを込めたいときには，人とか人間のように漢字で書くことにします．

ここで，ヒトの欲望の実現は，倫理や法律等に影響されない，と定義することにしましょう． {% sidenote %} これは定義であるよりも，我々をヒトとしたことの帰結であるように思える． それについての議論はここで． しかし，詳細は難しく，やや哲学的になるので，ここでは定義とすることに留めた． {% endsidenote %}

この定義から，我々ヒトの欲望が実現可能であるためには， それを欲し，それが原理的に可能であるだけで十分となります．

[imaOK 人間とヒトの実現可能性の条件の違いの図]

このことは，ヒトの世界が無法地帯になることを意味しません． 欲望と原理にのみ従った結果として，倫理や法律がなくても秩序が形成されることはよくあるからです．

例えばブロックチェーンではデータ改竄の試みは何の利益も生まないので，だれもデータを改竄しようとはしません． しかしそれは，データを改竄してはいけないという法律があることを意味しません． 単に，やりたくないから誰もやらないのです． つまりここでは， 望ましくない行為の利益を0にすることで，それをする欲望を消滅させ，法律や倫理なしで，秩序の形成に成功している のです．

[image:禁止しません　やりたくないのでいいです]

このような，欲望と原理に従った結果としてのルール形成・発展はヒトの範疇，ガイドの範疇となります． 一方，ヒトが欲して原理的にもできることを，たんに倫理や法律で禁止するという事態は人間の範疇なのでガイドでは考えることはありません．

{% endsubsection %} {% subsection "homo_sapienes","1.3|原理はより本質的な問題である",'1|原理的に可能な欲望はすべて許されている' %}

このように，倫理や法律など，原理以外の事項の検討を全くしない，という決断は正当化できるものでしょうか？

そこで，倫理を例に取って「これは原理的にできるか？」と「これは倫理的にすべきか？」の2つの問いのどちらが本質的なのかを考えてみましょう．

まずは普遍性の観点から．

「これは原理的にできるか？」の答えは，世界を支配する物理法則が変わらない限りは変わりません． 例えば，外部からエネルギーを与えることなく，運動し続ける機械をつくることは，原理的にできないので，いくら技術が進歩しようが，人間の考えが時間・場所によって，どう変わろうが，できません． 一方，「これは倫理的にすべきか？」の答えは，時代・地域によって変わります． 例えば，アリストテレスは「夫が妻に対して政治的支配を行うべきである」としました．(出典) 彼が生きた時代のギリシアではこの倫理観が支配的だったのでしょう． ですが，たった2000年後 もし我々が宇宙全体をコントロールしようと思うなら，4桁年という時間はとても短い． の現代の日本においては，これは全く間違っているとされています． しかし，「これは全く間違っている」という我々の意見に，例えば西暦5000年の地球人が同意してくれるのか，それを知る方法はありません． [imaOK OK倫理に欠けた西暦5000年の地球人] このように，「倫理的にすべきか？」という問いへの答えは，時空間の普遍性に欠けます．

次に優先順位の観点から．

「これは倫理的にすべきか?」の議論は，「これは原理的にできるか？」がyesの場合にのみ意味をもちます． そもそも原理的にできないことを，すべきか？と考えても意味がないからです． 例えば「時を戻すべきかどうか」を真剣に議論している人はいません． ここで注意してほしいのは「原理的にできるか？」は「今できるのか？」とは違うことです． 例えば，火星に移住することは今はできていませんが，原理的には実現可能です．ですから火星に移住すべきか？という倫理的議論は意味を持ちます． このように，原理は倫理に優先します．

普遍性から考えても，優先順位から考えても，より本質的なのは，「これは原理的にできるか？」という問いの方なのです． そこで，ガイドでは，原理と欲望の問題だけを扱うことにしました．

{% endsubsection %} {% subsection "homo_sapienes","1.4|ガイドで取り扱うヒトの欲望の例",'1|原理的に可能な欲望はすべて許されている' %}

最後に，ガイドで取り扱う，いくつかのヒトの欲望を見てみましょう．

  • 適切な気温・安全な場所で暮らしたい．
  • 肉体労働したくない，とくに農作物を自分で作りたくない．
  • 快楽を最大化するように意思決定したい．
  • 宇宙全体を支配したい．
  • 改造生物をつくりたい．
  • レーザーで星をぶっ壊したい

もし，これらを，あなたも欲するのなら，しばらく人間の世界とはお別れしましょう． ガイドにおいて，原理的に可能な欲望は，すべてが許されているのです．

{% endsubsection %} {% subsection "homo_sapienes","1.5|まとめ",'1|原理的に可能な欲望はすべて許されている' %}
  • 我々はヒトであるから，原理的に可能な欲望はすべて実現可能性がある
  • 人間であれば，原理以外の要素も実現可能性に関わる
  • ヒトになったのは，原理的に可能か？の問いがより本質的だからである
  • 欲望と原理だけから秩序は形成されうるが，これはガイドの範疇である
{% endsubsection %}
{% endsection %} {% section "homo_sapienes",'2|帰結から介入が予測でき，介入から帰結を求められることが，世界をコントロールすることである．' %}
{% subsection "homo_sapienes","2.1|欲望の実現方法が世界のコントロールの方法である",'2|帰結から介入が予測でき，介入から帰結を求められることが，世界をコントロールすることである．' %}

このガイドの目的は，

我々ヒトの欲望の実現のために世界をコントロールする方法を学ぶ (A)

でした．

前のセクションでは，「我々ヒトの欲望の実現のために」の意味を説明したので， このセクションでは，「世界をコントロールする」の意味を説明しましょう．

そもそも文Aで述べている通り，世界のコントロールはヒトの欲望の実現のために行います． そこで，ヒトの欲望の実現は，どのように行われるかを見ていくことにしましょう． それが「世界をコントロールする」の意味するところなはずです．

{% endsubsection %} {% subsection "homo_sapienes","2.2|欲望の実現方法は5つのステップで行われる",'2|帰結から介入が予測でき，介入から帰結を求められることが，世界をコントロールすることである．' %}

欲望の実現の第一のステップは，欲望を明晰にし，望ましい世界の姿について述べた文に変換することです．

例えば，適切な気温で暮らしたい，と欲望したとしましょう． しかし，この表現のままでは

  • 適切な気温とは何度なのか？
  • 暮らすとは何を意味しているのか？
などが明らかではありません．

これでは，どうすればこの欲望が実現できるか分からず，何をすれば良いかも分かりません． 欲望は目指すべき目標・ゴールなのですから，もっと明晰にしなければなりません．

[imaOKあいまいなことを言っているやつにきれる図]

そのためには，欲望を望ましい世界について述べた文にすることです． 例えば，適切な気温で暮らしたい，という欲望は， 「90%以上のヒトが20度の空間のなかで1日のうち20時間以上を過ごす」という望ましい世界についての文に変換できます．

こうして欲望を望む世界の形へ変換できれば，やるべきことはこのような世界の実現であり， 世界がこのような望ましい姿になった暁には，欲望は実現されたことになります．

やるべきことがはっきりしたところで， 第二のステップは「現在の世界が望ましい世界に変わる」という帰結のために，どんな介入が必要なのかを予測することです． 例えば，彼が怒る，という帰結となるためには，彼を殴ればいいだろう，と必要な介入を予測します．

[image:帰結から介入の予測　現在の世界と望ましい世界]

こうして，必要な介入を求めることができれば，あとは介入を実行するのみ，といきたいところですが， 欲望を叶えるために，ある介入を行うと，欲望の実現以外のことも起こるのが普通です．

例えば，彼を怒らせようとして，彼を殴れば，彼は確かに怒るかもしれませんが，彼は警察に通報して騒ぎになってしまうかもしれません． このような，介入による欲望の実現以外の帰結を．副作用と呼びましょう．

そうすると，介入による帰結は，欲望の実現+副作用ということになります． そこで我々は介入が与えられた時に，両者を合わせた全体の帰結を予測する必要があります．

例えば，彼を殴れば，彼は怒るだろうが，警察には通報しない，と介入による全体の帰結が予測できるなら，安心して彼を殴ることができます． 一方，もし，欲望の実現に比べて，副作用が望ましくないなら，その介入は行わず，別の介入を探すことになります．

以上から欲望を実現するステップをまとめると

  1. 欲望を明晰にし，望ましい世界についての文に変える
  2. 「現在の世界が望ましい世界に変わる」という帰結のために，どんな介入が必要なのかを特定する．
  3. その介入によるすべての帰結を予測する
  4. 欲望と副作用を比べて，欲望のほうが十分に望ましければ5へ．望ましくなければ2に戻って別の介入を探す．
  5. 介入を実際に行う．

ここで，セクション1において何度も出てきた「原理的に可能」の意味が明晰になります． すなわち，ステップ2において，もし，可能な介入が1つでもあれば，その欲望は原理的に可能で， 逆に1つも可能な介入が存在しなければ，その欲望は原理的に不可能ということになります．

また，ステップ5までたどり着いても，例えば技術的な理由で実際に介入が行えない場合は，その欲望は原理的には可能ではありますが，今はまだできないということになります． {% sidenote %} もしくは人間の場合は，倫理的・法的など色々な原因で実際の介入が行えない場合もあるだろう. {% endsidenote %}

{% endsubsection %} {% subsection "homo_sapienes","2.3|世界のコントロールは，ステップ2・3を主対象とする",'2|帰結から介入が予測でき，介入から帰結を求められることが，世界をコントロールすることである．' %}

欲望を実現するために我々ヒトが学ぶべきことを5ステップから抽出すると次のとおりです．

  1. 欲望の明晰化
  2. 帰結からの介入の予測
  3. 介入からの帰結の予測
  4. 欲望と副作用の価値判断
  5. 介入の技術的な実装
[image:5ステップの例]

世界をコントロールする方法を学ぶ，とはステップ2・3を中心に学ぶことを意味します．

というのも，まず，4の価値判断は，詳しく考えると，人間の領域に属するものだからです． ガイドでも価値判断はしますが，いくつかの欲望の直観的な好ましさを比べること以外は行いません．

5の技術的な実装に深く立ち入らない理由は，実装は時代によって変わるからです．

例えば，コンピュータを例に説明しましょう． 現代のコンピュータは半導体というもので作られています． しかし，半導体で作られていることはコンピュータにとって本質ではありません． というのも，コンピュータは昔は真空管で作られていましたし，将来的には，例えば光で作られることもあり得ます．

このように，コンピュータを何で作るか？という技術的な実装は時代とともに変わりうるものなのです． より本質的な問題，例えば，コンピュータは何を実現すればいいのか？は時代によって変化しません． 前者がステップ5に属し，後者がステップ2や3に属する問いであることを考えると，ガイドで2・3を中心に取り組むことは正当化できそうです．

以上の理由から，ガイドでは技術の細部には踏み込まず，実装の原理的部分や，コアとなるアイデアにフォーカスを当てます． といっても，例えば，

  • 岩から半導体を作る
  • 原油からあらゆる製品を作る
  • 鉄筋コンクリートを作る
など現代の代表的な技術については，実際のプロセスがイメージできるくらい詳しく扱う予定です．

最後に，ステップ1の明晰化は，世界をコントロールする方法そのものであるというよりは，世界をコントロールする方法を学ぶために，知っておく必要があることなのです． 例えば，四則演算は，連立方程式を解くために知っておく必要があるでしょう． それと同じように，明晰に考えられることは，世界をコントロールする方法を学ぶために，前提となることなのです．

そこで，明晰に考えることについては，tipsやchpater1など，世界をコントロールする方法を学ぶ前に，学んでおくことにしましょう． {% sidenote %} ただ，実際にはガイドで世界をコントロールする方法を学びながら，基本となる明晰に考える力を鍛えていくというふうになるだろう． {% endsidenote %}

以上から，世界をコントロールする方法を学ぶというとき，我々は，欲望の実現方法のうち，より原理的なステップ2と3を学ぶことを意味しています．

実現可能性において，倫理など本質的でない部分を捨てたように，世界をコントロールする，というときにも，価値判断や技術的詳細などには立ち入りすぎないことにします．

最後に，介入を「何もしないこと」としたときに，介入から帰結を予測することは，天気予報など，世界の成り行きを予測する問題となります． このような世界の未来の予測も，世界のコントロールの1例ですから，ガイドで学んでいくこととしましょう．
